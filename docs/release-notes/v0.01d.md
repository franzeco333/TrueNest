# TrueNest v0.01d - SVG import diagnosis correction

## English

This release focuses on correcting the SVG import diagnosis.

### Fixed

- Missing SVG `fill` is now treated as black by default.
- Black circles, rectangles and ellipses exported without explicit fill are detected correctly.
- Compound path detection remains available as possible holes/subpaths.

### Output

- TXT diagnosis report.
- HTML diagnosis report.

## Español

Esta versión corrige el diagnóstico de importación SVG.

### Corregido

- Si un SVG no declara `fill`, ahora se interpreta como negro por defecto.
- Círculos, rectángulos y elipses negros exportados sin fill explícito se detectan correctamente.
- Se mantiene detección de paths compuestos como posibles hoyos/subtrazos.

### Salida

- Reporte TXT.
- Reporte HTML.
