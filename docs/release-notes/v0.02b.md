# TrueNest v0.02b - Speed optimization

## English

This release focuses on performance.

### Changed

- Candidate position search was optimized.
- The beta now accepts the first valid position in priority order.
- Grid step increased from 2 mm to 4 mm for faster testing.
- Exact Shapely validation is still used before accepting any placement.

## Español

Esta versión se enfoca en rendimiento.

### Cambiado

- Se optimizó la búsqueda de posiciones candidatas.
- La beta acepta la primera posición válida según prioridad.
- La grilla pasó de 2 mm a 4 mm para acelerar pruebas.
- La validación exacta con Shapely se mantiene antes de aceptar posiciones.
