# TrueNest v0.03b - Direction suffixes and comparison outputs

## English

This release focuses on testing multiple nesting directions without overwriting files.

### Added

- Output suffixes based on direction and priority.
- Example: `_7_H` = bottom-left + horizontal.
- Larger buttons in the beta options window.
- Windows Notepad-friendly line endings.

## Español

Esta versión permite probar varias direcciones sin sobrescribir archivos.

### Agregado

- Sufijos según dirección y prioridad.
- Ejemplo: `_7_H` = abajo izquierda + horizontal.
- Botones más grandes en la ventana beta.
- Saltos de línea compatibles con Bloc de notas de Windows.
