# TrueNest v0.04 - Clean RDWorks output + PNG preview

## English

This release focuses on separating production output from visual debugging output.

### Added

- Clean SVG intended for RDWorks import.
- PNG preview image.
- TXT report in the export folder.
- Export folder organization.
- Center + diagonal behavior is exposed as center-style behavior.

### Changed

- Debug guides are not included in the production SVG.
- The work area is not exported by default.
- Preview/debug information belongs in the PNG, not the RDWorks SVG.

## Español

Esta versión separa la salida productiva de la salida visual de depuración.

### Agregado

- SVG limpio para importar en RDWorks.
- Imagen preview PNG.
- Reporte TXT en carpeta de exportación.
- Organización por carpeta de exportación.
- El comportamiento centro + diagonal se trata como comportamiento de centrado.

### Cambiado

- Las guías de depuración no se incluyen en el SVG productivo.
- La mesa no se exporta por defecto.
- La información visual queda en el PNG, no en el SVG para RDWorks.
