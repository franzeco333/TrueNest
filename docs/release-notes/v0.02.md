# TrueNest v0.02 - First real geometry nesting beta

## English

First beta of the real geometry nesting core.

### Added

- SVG import.
- Temporary red work area / black figure convention.
- Shapely validation for containment and collisions.
- Real spacing logic:
  - 3 mm configured spacing;
  - 1.5 mm figure offset;
  - 1.5 mm internal work area margin.
- SVG result output.
- TXT report.
